# Predictive Analytics Dashboard

## Overview

This project was developed as part of my Data Science Internship at Thiranex.

The dashboard uses historical sales data and Machine Learning techniques to forecast future sales trends and generate business insights.

## Features

* Historical Sales Analysis
* Sales Forecasting using Linear Regression
* Model Accuracy Evaluation
* Interactive Dashboard
* Trend Analysis and Visualization

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Plotly
* Streamlit

## How to Run

pip install -r requirements.txt

streamlit run app.py

## Project Outcome

This project helped me understand predictive analytics, forecasting techniques, trend analysis, and machine learning model evaluation.
